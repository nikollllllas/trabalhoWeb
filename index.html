Basic HTML

Doctype

A tag <!DOCTYPE> mostra qual o tipo do documento.

HTML

A tag <html> é a tag raiz de um documento HTML. É um container para todos as outras tags (menos a <DOCTYPE>). É nela que é definida a linguagem nativa do documento com o atributo "lang="linguagem"".

Head

A tag <head> funciona como um container para os metadados e informações sobre o documento, está presente entre as tags <html> e <body>. É onde se localiza também o título do documento (title), a tag link com valores de ícone do documento (favicon), estilo em cascata (css), scripts (javascript), etc.

Title

A tag <title> contém o título do documento, apresentado na aba junto ao favicon. No presente documento, o título é "Trabalho - Web".

Body

A tag <body> é o corpo do documento. Onde se encontra todo o conteúdo que será apresentado na tela. Só pode existir um <body> no documento.

H1...H6

As tags <h1> ... <h6> em suma são elementos de cabeçalho e definem o tópico da seção em que ele é apresentado. São divididos de H1 até H6, sendo H1 o mais relevante até H6 o de menor relevância. Aceita o atributo style dentro da tag de abertura para estilização.

Exemplos:

H1

H2

H3

H4

H5
H6

P

A tag <p> define um parágrafo, onde um texto é inserido na tela. Também aceita o atributo style dentro da tag de abertura para estilização.

Exemplo:

Este é um parágrafo.

E este outro parágrafo.

BR

A tag <br> insere uma quebra de linha (break = quebra | linha = row). Não necessita de fechamento. Pode ser utilizada para escrever poemas. Assim como
estas quebras de linha
feitas aqui.

Exemplo:
Aqui embaixo existe uma quebra de linha.
Aqui embaixo tem duas!

Pulou uma linha!

HR

A tag <hr> cria uma linha horizontal (horizontal = horizontal | row = linha) na tela para separar conteúdos. Não necessita de fechamento e aceita o atributo style dentro da tag para estilização.

Exemplo:

Uma horizontal line acima e uma abaixo!

Comentário

A tag <!--...--> é utilizada para "esconder" do navegador o código que estiver dentro dela.

Meta Info

Head

A tag <head> funciona como um container para os metadados e informações sobre o documento, está presente entre as tags <html> e <body>. É onde se localiza também o título do documento (title), a tag link com valores de ícone do documento (favicon), estilo em cascata (css), scripts (javascript), etc.

Meta

A tag <meta> define os metadados sobre a página. As tags <meta> são alocadas dentro do <head> e são utilizadas para especificar qual grupo de caracteres será aceito na página, descrição da página, autor e configurações de exibição da tela. Aceita os seguintes atributos: charset, content, http-equiv e name.

Base

A tag <base> define qual a URL base para todos os links relacionados no documento. A tag <base> precisa conter o atributo "href", que irá indicar qual o endereço base para todos os links de redirecionamento presentes na página.

Basefont

A tag <basefont> era utilizada no HTML4 para especificar cor da fonte, tamanho da fonte ou família da fonte. Hoje em dia é uma tag descontinuada (deprecated), pois, para a mesma função, usa-se as folhas de estilo (CSS).

Formatting

Acronym

A tag <acronym> foi descontinuada (deprecated). Era utilizada para sinalizar o significado de abreviações. Hoje em dia, utiliza-se a tag <abbr>.

Abbr

A tag <abbr> é utilizada para definir acrônimos como "HTML", "CSS", "Dr.", etc. As palavras envelopados com a tag <abbr> apresentam pontinhos abaixo delas. Ao repousar o mouse sob a palavra, o signifidado será apresentado. O atributo "title" é o responsável por apresentar o significado.

Address

A tag <address> define as informações de contato do autor/dono do documento/artigo. Nela pode conter endereço de e-mail, endereço físico, URL, número de telefone, redes sociais, etc.

Exemplos:

Praça Mascarenhas de Moraes
Site: wisdom.dog

B

A tag <b> é utilizada apenas em último caso, segundo o site MDN Web Docs, o elemento <b> não transmite semântica alguma, ou seja, só serve para deixar o texto em negrito.

Exemplo:
Uma palavra que precise de destaque!

BDI

A tag <bdi> é utilizada quando é necessário isolar uma parte do texto que é formatada de uma forma diferente de todo o resto. Dessa forma, entra em cena o isolamento bidirecional. Por exemplo: um árabe insere seu nome em seu idioma nativo, mas todo o resto do texto está formatado "normalmente" para nós do ocidente. A tag <bdi> faz com que a escrita árabe se comporte como estamos acostumados: escrita da esquerda para a direita. É útil quando o usuário está livre para inserir qualquer tipo de caractere. Não é suportada pelo Safari.

Exemplo: O usuário إيان fez 3 pontos.

BDO

A tag <bdo> é utilizada para inverter o texto completamente. Carrega consigo o atributo "dir", que aceita "rtl" e "ltr" o (padrão).

Exemplo:
Ao contrário.

big

A tag <big> era utilizada no HTML4 para aumentar o tamanho do texto que envelopava. Hoje em dia é uma tag descontinuada (deprecated), pois, para a mesma função, usa-se as folhas de estilo (CSS), mais especificamente, o atributo "font-size".

Blockquote

A tag <blockquote> especifíca uma citação de outra fonte. Por padrão, o navegador aplica a identação na citação. A tag aceita o atributo "cite", que recebe a url da fonte citada.

Exemplo:

Uma citação de algum livro muito interessante e logo, como um clássico ou algum artigo científico. Geralmente bem longa, assim como estamos aprendendo com a Ezilda em Metodologia da Pesquisa às quartas-feiras.

Center

A tag <center> não é mais utilizada no HTML5. Hoje, para centralização de elementos, utilizamos as folhas de estilo (CSS). O atributo utilizado é o "text-align: center".

Cite

A tag <cite> define o título de um trabalho criativo, seja ele uma música, livro, poema, pintura, etc. O texto envelopado com a tag é renderizado na tela em itálico.

Exemplo:
Don't Stay - Linkin Park
Master of Puppets (Remastered) - Metallica

Code

A tag <code> está sendo utilizada em todos os momentos em que tags são citadas neste documento. O texto envelopado com a tag é renderizado com uma fonte monoespaçada.

Del

A tag <del> define que um texto foi deletado do documento. Os navegadores irão renderizar o texto envelopado com um risco de ponta a ponta.

Exemplo:
Aqui nesse texto taxado.

DFN

A tag <dfn> é utilizada para indicar um termo que será definido posteriormente. A tag pode utilizar o atributo "title" ou melhor, a tag "abbr" usando o atributo "title".

Exemplo:
Neste documento foi utilizado o HTML, o CSS e um pouco de JS.

Em

A tag <em> está ligada a textos que necessitam de ênfase. O conteúdo envelopado é apresentado em itálico. Um leitor de tela lê com ênfase, modificando verbalmente a pronúncia.

Exemplo:
Texto com ênfase.

Font

A tag <font> foi descontinuada. Era utilizada para especificar a cor, família, tamanho, etc. Hoje em dia é utilizado o CSS para estilização.

i

A tag <i> define o texto envelopado como uma voz alternada e é renderizado em itálico. Geralmente utilizado para termos técnicos ou frases em outras línguas, um pensamento/idéia, nome de um navio, etc. A tag é utlizada quando não há nenhuma tag semântica mais apropriada como: "em", "strong", "mark", "cite" e "dfn".

Exemplo:
Navio: MSC SeaShore

Ins

A tag <ins> define que o texto envelopado foi inserido no documento (contrário da tag <del>, que define que um texto foi retirado do documento). É renderizado de forma sublinhada. Aceita os atributos "cite", indicando qual a razão do texto ser inserido/editado, e o atributo "datetime", que especifica a data que o texto foi inserido/editado.

Exemplo:
Leis LGPD LGBT

KBD

A tag <kbd> é usada quando for definida o uso do teclado para entrada. Assim como a tag <code>, todo o texto envelopado é renderizado como monoespaçado.

Exemplo:
Para copiar algo, selecione o texto desejado e utilize Ctrl + C.

Mark

A tag <mark> define o texto envelopado como importante. Assim como um marca-texto, define o fundo do texto envelopado como amarelo, podendo ser definidas outras cores com o CSS.

Exemplo:
Texto marcado!

Meter

A tag <meter> é utilizada para mostrar como uma escala um valor entre um intervalo conhecido ou valor fracionado. Aceita os atributos:

  • Value: valor númerico atual.
  • Min: valor mínimo.
  • Max: como valor máximo.
  • Low: por padrão é igual o valor de "min", é o limite numérico máximo da parte inferior do intervalo medido deve ser menor que o atributo "high".
  • High: por padrão é igual o valor de "max" e deve ser maior que o atributo "min" e menor que o atributo "max".
  • Optimum: o valor numérico ótimo. Deve estar entre os atributos "min" e "max". Quando utilizado com os atributos low e high, indidca a região que é preferível.
  • Form: esse atributo associa o meter à um form, que pode estar recebendo valores numéricos e o meter apresentando o valor correspondente ao que recebeu no input. O atributo pode ser omitido caso o elemento meter seja descendente de um form.

Exemplo de um meter com os atributos setados para:
Low: 60;
High 80;
Max: 100;
Optimum: 70;
e Value: 75;

Q

A tag <q> representa uma citação breve, na própria linha.

Exemplo:

RP e RT

A tag <rp> e a tag <rt> são utilizadas junto a tag <ruby>. A tag <rp> é responsável por prover parêntesis para um texto ruby, que assim irá conseguir renderizar para browsers que não tem suporte à notação do ruby. Já a tag <rt> é responsável por definir a explicação ou pronúncia de caracteres na notação do ruby.

Exemplo:

Ruby

A tag <ruby> especifíca que uma anotação ruby irá aparecer. É uma anotação de pequenos textos extras que ficam juntos com o texto principal indicando a pronúncia ou significado dos caracteres correspondentes. Esse tipo de anotação é muito utilizada em publicações japonesas.

S

A tag <s> quer dizer que o texto não está mais correto, relevante. O texto é renderizado com um risco horizontal do começo ao fim das palavras envelopadas com a tag. Não deve ser utilizado para definir textos deletados em um documento.

Exemplo: 

                            Queria estar em casa.
                            Que legal estar aqui!

Samp

A tag <samp> é utilizada quando o objetivo é mostrar que o texto foi emitido pelo computador ou por um software do mesmo.

Exemplo:
Error $%?!
Press F16 to continue.

Small

A tag <small> define que o texto envelopado será menor. Utilizado para mostrar direitos autorais e outros comentários secundários.

Exemplo:

sozinho.

Strike

A tag <strike> era utilizada no HTML4 para renderizar o texto envelopado com um risco horizontal (assim como <del> e <s> mas sem semântica alguma). Hoje em dia é uma tag descontinuada (deprecated), pois, como anteriormente, usa-se as tags <del> e <s>

Strong

A tag <strong> define a importância do texto envelopado. O texto é renderizado em negrito assim como na tag <b>.

Exemplo:
Importante!

Sub

A tag <sub> define textos subscritos. Aparecem meio caractere abaixo da linha padrão e pode ser renderizado em uma fonte menor.

Exemplo:
Água: H2O
Açúcar (Sacarose): C12H24O12
Açúcar em Cubos (Sacarose): (C12H24O12)3

Sup

A tag <sup> define textos sobrescrito. Ao contrário do subscrito, ele fica meio caractere acima da linha superior padrão e pode ser renderizado em uma fonte menor. Geralmente é utilizado em notas de rodapé.

Exemplos:
2x2 + 8y - 19a = 0
55 = 3125

O que está dentro desse container pode ser renderizado posteriormente com javascript. Time

A tag <time> é utilizada para definir um horário específico. O atributo datetime é um elemento utilizado para converter a hora em um formato que o navegador consegue interpretar, melhorando os mecanismos de pesquisa, gerando resultados de busca mais inteligentes. Não há nenhuma mudança visual.

TT

A tag <tt> foi descontinuada, ou seja, é considerada deprecated, no lugar dela usamos <kbd> para referenciar que um texto foi inserido via teclado, <var> para referenciar variáveis, <code> para linhas de código e <samp> para mostrar o texto emitido pelo computador.

U

A tag <u> representa que o texto é desarticulado e estilizado diferente do texto normal, com erros ortográficos. O conteúdo envelopado é renderizado com underlines.
